¿Qué es Arduino?

Arduino es una plataforma de electrónica de código abierto basada en hardware y software fáciles de usar. Fue creada en 2005 por Massimo Banzi y su equipo en el Instituto IVREA de Italia, con el objetivo de permitir a estudiantes, artistas, diseñadores y aficionados crear proyectos interactivos sin necesidad de conocimientos avanzados en electrónica.

Un Arduino es esencialmente un microcontrolador: una pequeña computadora en un chip capaz de leer señales del mundo físico (sensores, botones, temperatura) y controlar el mundo físico (LEDs, motores, pantallas). Todo esto se programa desde una computadora usando el IDE de Arduino, basado en el lenguaje C/C++.

¿Por qué aprender Arduino?
Arduino conecta el mundo digital con el mundo físico. Es la base del Internet de las Cosas (IoT), la robótica educativa y la automatización del hogar. En la especialidad de Inteligencia Artificial, Arduino permite crear dispositivos físicos que recopilan datos del entorno para alimentar sistemas de IA.

La Placa Arduino UNO

El Arduino UNO es el modelo más popular y el mejor para comenzar. Está basado en el microcontrolador ATmega328P y cuenta con todos los elementos necesarios para empezar a crear proyectos electrónicos de inmediato.

Los Pines de Arduino

Los pines son los puntos de conexión de la placa con el mundo exterior. Existen tres tipos principales:

⚡ Pines Digitales (0–13)

Solo leen o envían dos valores: HIGH (5V) o LOW (0V). Sirven para controlar LEDs, relés y leer botones. Los marcados con ~ (3, 5, 6, 9, 10, 11) soportan PWM para simular valores analógicos.

📊 Pines Analógicos (A0–A5)

Leen voltajes entre 0 y 5V y los convierten a un número entre 0 y 1023 (resolución de 10 bits). Ideales para sensores de temperatura, luz, distancia y potenciómetros.

🔋 Pines de Alimentación

5V y 3.3V: suministran voltaje a componentes externos. GND: tierra (referencia de 0V), siempre se conecta con los componentes. Vin: entrada de voltaje externo (igual que el jack de alimentación). AREF: referencia de voltaje para las entradas analógicas.

Componentes Electrónicos Esenciales

Además de la placa Arduino, estos son los componentes que se usan en la mayoría de los proyectos básicos:

LED

Diodo emisor de luz. Es el componente más básico: enciende, apaga y parpadea para indicar estados. Siempre requiere una resistencia en serie para no quemarse.

Salida Digital
Resistencia

Limita el paso de corriente. Se mide en Ohms (Ω). Sus bandas de colores indican su valor. Protege componentes como LEDs y sensores de sobrecorriente.

Componente Pasivo
Protoboard

Tablero de prototipado sin soldadura. Permite conectar componentes temporalmente con cables Dupont. Sus filas y columnas comparten conexión internamente.

Prototipado
Botón Pulsador

Interruptor que cierra el circuito mientras se presiona. Se lee como HIGH o LOW en un pin digital. Requiere resistencia pull-down o pull-up para evitar lecturas flotantes.

Entrada Digital
Sensor de Temperatura

El LM35 y DHT11 son los más comunes. Convierten la temperatura del ambiente en un voltaje o señal digital que Arduino lee con analogRead().

Entrada Analógica
Sensor Ultrasónico

El HC-SR04 mide distancias emitiendo pulsos de sonido y midiendo el tiempo que tarda el eco en regresar. Rango de 2 cm a 400 cm con gran precisión.

Entrada Digital
Servomotor

Motor de posición controlado por señal PWM. Gira entre 0° y 180°. La librería Servo.h de Arduino hace muy sencillo su manejo para brazos robóticos y más.

Salida PWM
Potenciómetro

Resistencia variable ajustable con una perilla. Al girar, cambia el voltaje de salida entre 0 y 5V. Se lee con analogRead() y se usa para controlar brillo, velocidad, volumen.

Entrada Analógica

Tu Primer Programa: Blink

El "Hola Mundo" de Arduino es el programa Blink: hacer parpadear el LED integrado del pin 13. La estructura de todo programa Arduino se divide en dos funciones obligatorias:

blink.ino 
// ── PROGRAMA BLINK — LED parpadeando ────────────────────

// Definir una constante para el pin del LED
const int LED_PIN = 13;

// setup(): se ejecuta UNA sola vez al encender o resetear
void setup() {
  // Configurar el pin 13 como SALIDA
  pinMode(LED_PIN, OUTPUT);
}

// loop(): se ejecuta en bucle infinito mientras Arduino esté encendido
void loop() {
  digitalWrite(LED_PIN, HIGH);  // Encender LED (5V)
  delay(1000);                   // Esperar 1000 ms (1 segundo)

  digitalWrite(LED_PIN, LOW);   // Apagar LED (0V)
  delay(1000);                   // Esperar 1 segundo
}
sensor_temperatura.ino 
// ── LEER SENSOR DE TEMPERATURA LM35 ─────────────────────
const int SENSOR_PIN = A0;   // Pin analógico A0

void setup() {
  Serial.begin(9600);         // Iniciar comunicación serial
}

void loop() {
  int lectura   = analogRead(SENSOR_PIN);   // 0–1023
  float voltaje  = lectura * (5.0 / 1023.0);   // convertir a voltios
  float tempC    = voltaje * 100.0;            // LM35: 10mV por °C

  Serial.print("Temperatura: ");
  Serial.print(tempC);
  Serial.println(" °C");

  delay(500);
}

Funciones Clave del IDE de Arduino

digitalRead(pin)Lee el estado (HIGH/LOW) de un pin digital.
analogRead(pin)Lee el voltaje en un pin analógico. Devuelve 0–1023.
analogWrite(pin, val)Envía señal PWM (0–255) en pines compatibles (~).

Guía para Empezar: Paso a Paso

1

Instala el IDE de Arduino

Descarga el Arduino IDE desde arduino.cc/en/software. Es gratuito y disponible para Windows, Mac y Linux. También puedes usar el Arduino Web Editor en línea sin instalar nada.

2

Consigue los componentes básicos

Empieza con un kit básico: placa Arduino UNO, protoboard, cables Dupont, LEDs (varios colores), resistencias (220Ω y 10kΩ), un botón pulsador y un potenciómetro. Costo aproximado: $200–$350 MXN.

3

Carga el programa Blink

Abre el IDE → Archivo → Ejemplos → 01.Basics → Blink. Conecta tu Arduino, selecciona el puerto COM correcto y presiona el botón de cargar (→). El LED del pin 13 debería comenzar a parpadear.

4

Experimenta con la protoboard

Conecta un LED externo con su resistencia de 220Ω entre el pin 13 y GND en la protoboard. Cambia el tiempo del delay() para modificar la velocidad del parpadeo y observa el resultado.

5

Agrega sensores y el Monitor Serial

Conecta un potenciómetro al pin A0 y usa analogRead() + Serial.println() para ver los valores en tiempo real en el Monitor Serial (Ctrl+Shift+M). Así verás cómo Arduino lee el mundo físico.

6

Explora proyectos y la comunidad

Plataformas como Tinkercad Circuits (simulador gratuito online), Instructables y el foro oficial de Arduino tienen miles de proyectos con tutoriales completos para todos los niveles.

Conexión con Inteligencia Artificial: Arduino puede comunicarse con una computadora o Raspberry Pi vía serial o WiFi, enviando datos de sensores que alimentan modelos de IA en Python. Proyectos como detección de gestos, monitoreo ambiental inteligente y control por voz son posibles combinando Arduino con librerías de IA.
