Desarrollo Web

H
HTML5 Marcado

HyperText Markup Language es la estructura fundamental de toda página web. Define el contenido y la semántica del documento mediante etiquetas.

index.html 
<!DOCTYPE html>
<html lang="es">
  <head>
    <title>Mi Página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Bienvenido a HTML5</p>
  </body>
</html>
C
CSS3 Estilos

Cascading Style Sheets controla la presentación visual de las páginas web: colores, tipografía, distribución y animaciones.

estilos.css 
.tarjeta {
  background: #1a9ed4;
  border-radius: 12px;
  padding: 20px;
  transition: all .3s ease;
}
.tarjeta:hover {
  transform: translateY(-6px);
}
JS
JavaScript Scripting

El lenguaje de programación del navegador. Añade interactividad, lógica y dinamismo a las páginas web. Hoy también se usa en servidores (Node.js).

app.js 
// Función de bienvenida
function saludar(nombre) {
  const msg = `Hola, ${nombre}!`;
  console.log(msg);
  return msg;
}

saludar("Marcos");
PHP
PHP Backend

PHP es el lenguaje del servidor más usado en la web. Potencia WordPress, Facebook (en sus inicios) y millones de sitios. Ideal para lógica de negocio y bases de datos.

index.php 
<?php
// Conexión y consulta
$nombre = "Marcos";
$año    = 2026;

echo "Bienvenido, $nombre";

if ($año >= 2026) {
  echo "¡Año actual!";
}
?>
TS
TypeScript Tipado

Superconjunto de JavaScript desarrollado por Microsoft que añade tipado estático, mejorando la detección de errores en proyectos grandes.

usuario.ts 
interface Usuario {
  nombre: string;
  edad:   number;
}

function bienvenida(
  u: Usuario
): string {
  return `Hola ${u.nombre}`;
}
SQL
SQL Base de datos

Structured Query Language es el estándar para gestionar bases de datos relacionales. Todo sistema web que almacena datos utiliza SQL o sus variantes.

consulta.sql 
-- Crear tabla de alumnos
CREATE TABLE alumnos (
  id      INT PRIMARY KEY,
  nombre  VARCHAR(100),
  grado   INT
);

SELECT * FROM alumnos
WHERE grado = 3;

Desarrollo de Escritorio

Py
Python General

Lenguaje multiparadigma conocido por su sintaxis limpia. Domina en IA, ciencia de datos, automatización y también en desarrollo de apps de escritorio con Tkinter o PyQt.

app.py 
# Interfaz gráfica simple
import tkinter as tk

ventana = tk.Tk()
ventana.title("Mi App")

btn = tk.Button(
  text="Hola Mundo"
)
btn.pack()
ventana.mainloop()
Ja
Java POO

Uno de los lenguajes más usados del mundo. Orientado a objetos, multiplataforma ("escríbelo una vez, ejecútalo en cualquier lugar"). Base del desarrollo Android y de apps empresariales.

Hola.java 
public class Hola {
  public static void
  main(String[] args) {

    System.out.println(
      "¡Hola Mundo!"
    );
  }
}
C#
C# Microsoft

Lenguaje de Microsoft usado con el framework .NET para apps de escritorio en Windows, videojuegos con Unity y servicios en la nube con Azure.

Program.cs 
using System;

class Program {
  static void Main() {
    string nombre =
      "Marcos";
    Console.WriteLine(
      $"Hola {nombre}"
    );
  }
}
Sw
Swift Apple

Creado por Apple para desarrollar apps en iOS, macOS, watchOS y tvOS. Moderno, seguro y veloz; reemplazó a Objective-C como el lenguaje nativo del ecosistema Apple.

ContentView.swift 
import SwiftUI

struct ContentView: View {
  var body: some View {
    Text("Hola Mundo")
      .padding()
      .foregroundColor(
        .blue
      )
  }
}
C++
C++ Sistemas

Lenguaje de alto rendimiento para sistemas operativos, videojuegos (Unreal Engine) y software donde la velocidad es crítica. Extensión de C con programación orientada a objetos.

main.cpp 
#include <iostream>
using namespace std;

int main() {
  int x = 10;
  cout <<
    "Valor: " << x
    << endl;
  return 0;
}

Diseño & Maquetación

Ss
Sass / SCSS Preprocesador

Extensión de CSS que añade variables, anidamiento, mixins y funciones. Permite escribir hojas de estilo más organizadas y reutilizables, compilándose a CSS estándar.

estilos.scss 
$primario: #1a9ed4;
$radio:    12px;

.card {
  background: $primario;
  border-radius: $radio;

  &:hover {
    opacity: .85;
  }
}
SVG
SVG Vectorial

Scalable Vector Graphics: formato XML para gráficos vectoriales en la web. Los íconos, logotipos e ilustraciones SVG escalan sin perder calidad a cualquier tamaño.

icono.svg 
<svg
  viewBox="0 0 100 100"
  xmlns="…">

  <circle
    cx="50" cy="50"
    r="40"
    fill="#1a9ed4"/>

  <text x="50" y="55"
    text-anchor="middle">
    Web
  </text>
</svg>
LTX
LaTeX Tipografía

Sistema de composición tipográfica de alta calidad utilizado en documentos académicos, tesis, libros científicos y publicaciones con fórmulas matemáticas.

documento.tex 
\documentclass{article}
\title{Cultura Digital}
\author{Marcos E.}

\begin{document}
  \maketitle
  % Fórmula matemática
  \( E = mc^2 \)
\end{document}
Md
